# 📘 Revisão de Python

Este projeto reúne minha trajetória revisando os fundamentos de Python, com foco em prática e consolidação do aprendizado.

Ao longo do desenvolvimento, explorei diferentes exercícios e desafios para reforçar a lógica de programação e ganhar mais familiaridade com a linguagem.

## 🚀 Objetivo

Relembrar conceitos essenciais e fortalecer a base para projetos mais avançados no futuro.
